# ☕ Coffee Shop Project

## 📌 Overview
This project is a Coffee Shop Management System / Data Analysis project designed to handle and analyze coffee shop operations such as orders, sales, and customer behavior.

## 🚀 Features
- Manage orders and products
- Track daily sales
- Analyze customer behavior
- Generate basic reports and insights

## 🛠️ Technologies Used
- Python / SQL / Pandas / Power BI (حسب مشروعك)
- Jupyter Notebook / VS Code

## 📊 Project Goal
The goal of this project is to practice data analysis / backend development skills and build a real-world business use case.

## 📁 Files Included
- dataset.csv
- analysis.ipynb
- queries.sql (if exists)
